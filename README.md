# 🛒 Shopping Project

A simple shopping web application built using **HTML, CSS, and JavaScript**. The project fetches product data from an API, displays products, shows product details, and includes a shopping cart with bill summary.

## 🚀 Live Demo

https://shopping-project-ebon.vercel.app

## 📂 GitHub Repository

https://github.com/soni-701/shopping-project

## ✨ Features

- Display products from DummyJSON API
- Product details page
- Add products to cart
- Remove items from cart
- Bill summary
- Responsive design
- Clean and simple UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DummyJSON API
